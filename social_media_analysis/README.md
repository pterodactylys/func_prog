# Анализатор социальных сетей

Приложение для сбора текстов из Reddit и VK и подсчёта базовой статистики по словам и хештегам. Работает на Python.

## Настройка учетных данных API

Создайте файл `.env` в корневой папке проекта или установите переменные окружения через настройки системы.

Необходимые переменные:

- Reddit
  - `REDDIT_CLIENT_ID`
  - `REDDIT_CLIENT_SECRET`
  - `REDDIT_USER_AGENT` (любая короткая строка)
- VK
  - `VK_TOKEN` (сервисный или пользовательский токен с доступом к стенам публичных групп)

Пример `.env`:

```
REDDIT_CLIENT_ID=xxxxxxxx
REDDIT_CLIENT_SECRET=yyyyyyyy
REDDIT_USER_AGENT=app/1.0
VK_TOKEN=your_token_here
```

## Установка зависимостей

```
pip install -r requirements.txt
```

## Запуск GUI

```
python3 main.py
```

- Вкладка Reddit: введите названия сабреддитов (через запятую), выберите лимиты, опционально сохранение в SQLite.
- Вкладка VK: введите числовые ID групп (через запятую, без ведущего минуса), выберите лимиты, опционально сохранение в SQLite.
- Кнопка «Start Both» запускает сбор из Reddit и VK в параллельных потоках.

## Результаты и сохранение данных

- Результаты (топ слов и хештегов) отображаются в интерфейсе.
- Если включена опция «Save to SQLite», данные записываются в две таблицы: `topics(word,count)` и `hashtags(tag,count)` в выбранный файл базы данных.
